# Aventura no Labirinto

Um jogo de console modular desenvolvido em Python.

## Como Executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o jogo: `python main.py --name "SeuNome" --dificuldade 1`

## Comandos
- Setas do teclado: Movimentação
- Coletar $: Ganha pontos
- Chegar no E: Vitória
